КРАТКОЕ ОПИСАНИЕ:
Этот проект представляет собой сайт стоматологической клиники, разработанный в рамках учебной задачи. 
Сайт состоит из нескольких страниц, и его цель - предоставить информацию о клинике, услугах, и контактных данных для 
потенциальных пациентов.

ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

*Главная страница: содержит краткую информацию о клинике, оказываемых услугах, о месторасположении и контактах,
а также отзывы пациентов.

*Услуги: на этой странице перечислен пример услуги "лечение кариеса" с краткой информацией, стоимостью.

*Врачи: карточки с данными и изображениями врачей.

ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ:

*HTML и CSS: для создания структуры и визуального оформления сайта.

* Figma: Макет сайта был разработан в Figma, и все дизайн-решения были взяты из макета, включая цветовую палитру и шрифты.
Все необходимые изображения были скачаны из макета в Figma и использованы на сайте.

* Адаптивная верстка: Сайт разработан с учетом адаптивности для мобильных устройств и планшетов. 
Использованы контрольные точки для обеспечения корректного отображения на разных экранах.

* Макет сайта был разработан с использованием 12-колоночной модульной сетки, что позволяет выравнивать элементы по сетке 
для улучшения дизайна и удобства пользователя.

### ИНСТРУКЦИИ ПО УСТАНОВКЕ:

1. Склонируйте репозиторий на свой компьютер.
2. Откройте `index.html` в вашем веб-браузере.
